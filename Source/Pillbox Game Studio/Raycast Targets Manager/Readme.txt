To use this utility just select any object you would like to inspect, then:
- Right click on selected object -> Manage Raycast Targets
or
- Open "Game Object" menu -> Manage Raycast Targets

Tool is campatible with any components which inherits unity's Graphic class, including TextMeshPro.
